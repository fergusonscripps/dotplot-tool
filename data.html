Mark epitope responses for each timepoint/donor. Use the checkboxes to indicate presence and partial responses.
How to mark responses:
✓ = Response present
½ = Partial response (only available when ✓ is checked)
No checkboxes → No dot shown
✓ only → Full solid dot
✓ + ½ → Partial dot (donut ring or open circle)
